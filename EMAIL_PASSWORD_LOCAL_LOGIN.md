# HELM — تشغيل الدخول المحلي بالإيميل وكلمة المرور

## ما أضيف في هذه النسخة

1. زر دخول Google كما هو.
2. نموذج دخول بالبريد الإلكتروني وكلمة المرور.
3. إنشاء حساب جديد بالبريد الإلكتروني.
4. رابط إعادة تعيين كلمة المرور.
5. صفحة عامة للمكتبة القانونية بدون تسجيل دخول:
   - `/PublicLegalLibrary`
6. عرض مختصر للمكتبة القانونية داخل صفحة الدخول نفسها.

## خطوات Supabase المطلوبة

1. افتح Supabase Dashboard.
2. ادخل إلى Authentication ثم Providers.
3. فعّل Email Provider.
4. إذا أردت دخولًا محليًا سريعًا أثناء التطوير، يمكنك تعطيل Confirm email مؤقتًا.
5. نفذ ملف SQL التالي من Supabase SQL Editor:

```sql
EMAIL_PASSWORD_AUTH_SETUP.sql
```

## إنشاء مستخدم إداري بالإيميل

بعد إنشاء المستخدم من صفحة الدخول أو من Supabase Auth، اجعل حسابك Admin بتنفيذ:

```sql
update public.user_profiles
set role = 'admin', full_name = 'المستشار أحمد حلمي'
where lower(email) = lower('ضع_الإيميل_هنا');
```

## ملاحظة مهمة

المكتبة القانونية العامة لا تفتح بيانات العملاء أو القضايا أو الفواتير. هذه البيانات تظل خلف تسجيل الدخول والصلاحيات.
